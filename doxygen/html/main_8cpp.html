DESCRIPTION

Source code to obtain target velocity using PID controller.

The default gain values are Kp=0.15, Ki=0.05, Kd=0.02. The user has option of changing these gain values if needed. The code takes in initial and target velocities from the user and for time step of 0.1 sec the program calculates the error and uses the PID controller to increase its velocity until the target velocity is reached.
